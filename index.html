About Us

WHO WE ARE: A close community of girls at Stuyvesant High School passionate about computer science. OUR MISSION: Provide a supportive and fun environment to learn computer science and help close the gender gap in computer science.

OUR VALUES:

[3] CSS 1 | 10/28/22

Today we continued to learn CSS with Ankita and Eva. First, we reviewed CSS mentioned in our previous lesson such as inline style, external style, selectors, properties, and the box model. There are four different kinds of selectors: group, class, ID, and universal. Common properties are font attributes and units. Then, we answered a review question with whiteboards. We learned about sizing images in terms of pixels and percentages. A set number of pixels stays constant despite screen size and a set percentage changes the image depending on the screen size. We also learned about border-radius which changes how round the edges of an image will be. In addition, we learned about flexbox. Flexbox specifies the way elements will be laid out on a webpage. Then, we did a short activity with Flexbox Froggy. After the activity, we learned about keyframes. Keyframes are used to create animations on your website for text, images, buttons and more. The syntax of keyframes is @keyframes {}. Then, Ankita and Eva displayed an example of an animation on the text, ‚Äúhello world‚Äù. Lastly, we learned about positioning which controls how specific elements are displayed and positioned on the website. The work time activity was creating and styling Halloween animations! Happy (early) Halloween!
